# AgentCanary — Market Intelligence Skill for AI Agents

Cross-asset market intelligence API for AI agents. 130+ endpoints across macro regime detection, risk scoring, trading signals, DeFi intelligence, BTC options flow, geopolitical risk, institutional positioning, and 4× daily AI market briefs.

## Install

```bash
# OpenClaw / ClawHub
clawhub install MrCerq/agentcanary

# npx skills (skills.sh compatible)
npx skills add MrCerq/agentcanary-skill

# Manual
Copy SKILL.md and references/ to your agent's skills directory.
```

## What It Does

- **Macro Regime Detection** — Risk-On/Off/Neutral with risk gauge scoring
- **Trading Signals** — IGNITION/ACCUMULATION/DISTRIBUTION/CAPITULATION states
- **DeFi Intelligence** — P/E ratios, token unlocks, yields, stablecoin flows
- **BTC Options** — Max pain, skew, put/call ratio, DVOL
- **Geopolitical Risk** — Conflict monitoring with commodity impact mapping
- **4× Daily Briefs** — AI-generated market intelligence (Radar, Signal, Pulse, Wrap)
- **Whale Alerts** — Real-time large crypto transaction monitoring
- **Institutional Positioning** — 13F filings, short interest, CFTC COT

## Auth

Wallet-based API keys. USDC/USDT on any major EVM chain (Base, Ethereum, Arbitrum, Optimism, Polygon). No KYC. No subscriptions. Credits never expire.
